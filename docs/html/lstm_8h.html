Detailed Description

LSTM functionalities [Perhaps most interesting from a user perspective].

Here are some functions that define, produce and use the LSTM network.

A LSTM network consists of an array of lstm_model_t data.

Each model in this array is called a layer. Output is collected in the beginning of the array, and input is entered in the end.

Say there is L > 0 layers, in a model defined as lstm_model_t ** model, then model[0] refers to the output layer and model[L-1] refers to the input layer.
